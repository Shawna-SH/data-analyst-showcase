# Data Analyst Showcase

This repository contains the source code for my interactive data analyst portfolio website.

The site showcases selected projects in data analytics, data engineering, and machine learning, with interactive demos and project summaries designed to highlight practical problem-solving and technical skills.

## Live Site

[Ruipu Shi_Protfolio](https://data-analyst-showcase--shiruipu23.replit.app)

## Featured Project Types

The portfolio includes examples of:

- Data analysis and exploratory data analysis (EDA)
- Data pipelines and data processing workflows
- Machine learning models
- Analytics dashboards and visualisations
- Interactive demos for model inference

## Tech Stack

Frontend:
- React
- TypeScript
- Vite
- Tailwind CSS

Data / ML demos:
- Python
- Machine learning inference APIs

## Purpose

This portfolio is designed to demonstrate:

- Practical data analysis workflows
- End-to-end project thinking
- Clear communication of technical work
- Interactive demonstrations of analytical tools
